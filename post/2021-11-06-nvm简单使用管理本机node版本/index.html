nvm简单使用（管理本机node版本）

使用 Go 编写的 Windows 下的 Node.js 版本管理工具

Posted by LP's BLOG on Saturday, November 6, 2021

一、 安装

下载对应的安装包nvm-setup.zip（这里以windows为例）

直接安装到eg: C盘根目录(路径不能有空格和中文即可)

二、使用镜像

目录下的settings.txt添加镜像（保证install node成功）

node_mirror: https://npm.taobao.org/mirrors/node
npm_mirror: https://npm.taobao.org/mirrors/npm

三、使用nvm

右键系统管理员方式打开cmd

nvm install [node版本]
nvm list 或者 nvm ls
nvm use [node版本]

假如步骤正确而始终报错，可用重启大法试一试
