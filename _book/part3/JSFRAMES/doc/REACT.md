# REACT

## 简介

> React 是一个声明式，高效且灵活的用于构建用户界面的 JavaScript 库。使用 React 可以将一些简短、独立的代码片段组合成复杂的 UI 界面

## 搭建环境

1. 安装nodejs
2. 创建react新项目

    ```s
    npx create-react-app my-app
    ```

3. 删除新项目中src中的所有源文件
4. 导入react包

    ```js
    import React from 'React'
    import ReactDOM from 'react-dom'
    ```

5. 运行

    ```s
    yarn start
    ```

1. Vue 与 React 的区别？
